# To Do Web API

This is an Web API to keep track of a list of pending tasks, and to be able to mark them as done.

## Clone

Clone the repository:
```bash
git clone https://github.com/lautarogalante/to-do.git
cd to-do
```

## Install the dependencies
```bash
dotnet restore
```

## Run API app

```bash
dotnet run
```
